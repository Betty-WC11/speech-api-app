# Motivation Booster! AAC Project

This is a React based communication aid or AAC (Augmentative and Alternative Communication) tool for those who want to spread positive and motivational energy to others.

<br>

The user can click on the movitvational quote and the application will use speech to communicate that quote.

![screenshot of tool](./src/assets/Screenshot%20UI.png)

The user can also select a preferred voice.

![screenshot of voice list](./src/assets/Screenshot%20voicelist.png)

## About the Tool

The tool is built using React JS with the incoporation of Web Speech API. It was a simple project to explore with Web Speech API and how to implement that into a React application.

## Getting Started

To start please make sure you have:

- Node.js installed on your machine
- npm or yarn package manager

**Installation**

1. Clone the repository

```
bash

git clone https://github.com/Betty-WC11/speech-api-app.git

cd speech-api-app
```

2. Install dependencies

```
bash

npm install
```

3. Run the development server

```
bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
